Métodos HTTP: 	GET (listar registros),
			 	POST (criar/salvar registro), 
				PUT (editar registro),
				DELETE (deletar registro)
Query Params: request.query -> filtros
Route Params: request.params -> adicionar ou deletar
Body: request.body -> criação ou alteração de registro

Routes:	INDEX (listar registros)
		SHOW (mostrar um registro)
		STORE (criar um registro)
		UPDATE (alterar um registro)
		DESTROY (deletar um registro)

MongoDB (não-relacional)
